/*! \page std_names Standard names of NetCDF variables, including PISM current and proposed \section existing Names from CF 1.4 Standard Name Table for land ice and/or currently used in PISM This list a small subset of the CF 1.4 standard name table. The alphabetical list here includes \li those with "land_ice" in the name and/or \li those used by Parallel Ice Sheet Model (PISM). If the "PISM short name" column is blank then PISM does not use that standard_name. Some of the short names are shared with GLIMMER and CISM. The names below starting with "land_ice" are believed to have all been submitted by Magnus Hagdorn to the CF committee circa 2003. Like the legendary Eskimo, the CF standard name table has a bazillion names for snow "amounts" and "fluxes". No attempt at using those names is made in PISM. Readers other than PISM programmers should ignore the final column, which gives the name of the IceModel member which holds the named field in an IceModelVec. The names starting with "PAC." and "POC." are exceptions because they are not IceModel members but instead they are members of classes derived from PISMClimateCoupler. Note PAC= PISMAtmosphereCoupler and POC= PISMOceanCoupler.
